# Fuel-Management-System

🚀 Excited to share my latest project: Fuel Management System (Java Swing)

I designed and developed a desktop-based application using Java Swing that simulates a real-world fuel billing system.

🔹 Key Features:
• Fuel selection (Petrol, Diesel, CNG) with dynamic input handling
• Automatic bill calculation based on quantity and rate
• Dropdown-based unit and pricing system
• Professionally formatted receipt-style bill generation
• Bill history management (Add, Update, Delete)
• Data persistence using file handling

🔹 What I Learned:
• Java Swing GUI design & layout management
• Event-driven programming
• Input validation using DocumentFilter
• File handling in Java
• Structuring real-world applications

This project gave me hands-on experience in building interactive desktop applications and strengthened my core Java skills.
